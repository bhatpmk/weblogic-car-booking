#  Sample application built with MicroProfile AI CDI extensions for WebLogic 15.1.1

## Introduction

The application in the repository demonstrates leveraging LLM capabilities in enterprise java applications. The application
is built using [MicroProfile CDI extensions for Langchain4J](https://github.com/langchain4j/langchain4j-cdi) and
using Jakarta EE 9.1 dependencies.

## Run the examples

For build and test instructions, see the [README for the car-booking sample](sample/README.md).
